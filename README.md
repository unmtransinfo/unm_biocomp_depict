# `UNM_BIOCOMP_DEPICT`

Molecular depiction classes, including servlets.

## Dependencies

* Java 8
* Maven 3.5+
* ChemAxon (19.3.0)
* Access to [ChemAxon Maven repository](https://hub.chemaxon.com)
(see [documentation](https://docs.chemaxon.com/display/docs/Public+Repository))
  * Requires API key.
* `unm_biocomp_util`

## Compiling

```
mvn clean install
```
